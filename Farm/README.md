# 🌾 乡村农场大亨 (Rural Farm Tycoon)

一个基于JavaScript的农场经营模拟游戏，玩家需要在7年时间内（2024-2030）通过科学的农业管理实现收益最大化。

## 🎮 游戏特色

### 核心系统
- **土地管理系统**: 1201亩土地，34个不同类型地块，16个大棚设施
- **作物种植系统**: 7种作物，轮作制度，豆类轮种要求
- **经济系统**: 双重销售模式，动态价格波动
- **随机事件系统**: 天气变化，市场波动，政策影响
- **数据分析**: 历史数据追踪，市场预测，收益优化

### 游戏机制
- **轮作限制**: 同一地块不能连续种植相同作物族
- **豆类轮种**: 每个地块3年内必须种植一次豆类作物
- **销售策略**: 直销模式vs降价出售，影响最终收益
- **设施管理**: 普通大棚vs智慧大棚，不同效率和成本

## 🎯 游戏目标

在7年经营期间（2024-2030）实现以下目标：
- 最大化总利润和农场可持续发展
- 合理规划作物轮作，保持土壤健康
- 应对市场波动和随机事件挑战
- 优化设施投资和资源配置

## 🌱 作物类型

### 粮食作物
- **小麦** 🌾: 价格稳定上涨5-10%/年，适合大规模种植
- **玉米** 🌽: 高产量，市场需求稳定
- **水稻** 🌾: 需要水洼地，价格相对稳定

### 豆类作物 
- **大豆** 🫘: 用于轮作，改善土壤，价格波动较大

### 蔬菜作物
- **白菜** 🥬: 适合大棚种植，季节性需求
- **胡萝卜** 🥕: 生长周期短，收益稳定
- **番茄** 🍅: 高价值作物，大棚种植效果好

## 🏗️ 设施系统

### 地块类型
- **平原地**: 适合多种作物，土壤肥沃
- **耕田**: 传统农田，适合粮食作物
- **山坡地**: 排水良好，适合小麦和大豆
- **水洼地**: 适合水稻和部分蔬菜

### 大棚设施
- **普通大棚**: 效率120%，年维护费1000元，可种植蔬菜和粮食
- **智慧大棚**: 效率150%，年维护费2000元，专门种植蔬菜

## 📊 市场机制

### 价格体系
- **小麦**: 年增长5-10%，长期投资价值高
- **其他作物**: 年波动±5%，需要关注市场时机
- **销售模式**: 
  - 直销: 按预期价格销售
  - 降价: 超出部分按50%价格出售

### 随机事件
- **天气事件**: 干旱、洪涝、高温影响产量
- **市场事件**: 需求激增、市场崩盘影响价格
- **成本事件**: 燃料、化肥价格波动
- **政策事件**: 政府补贴、新技术推广

## 🎮 操作指南

### 基础操作
1. **选择作物**: 在种植管理面板选择要种植的作物
2. **点击地块**: 在农场画面点击空闲地块进行种植
3. **查看信息**: 点击地块查看详细信息和种植状态
4. **收获作物**: 作物成熟后点击收获按钮

### 界面说明
- **农场视图**: 左侧大画布显示农场全貌
- **种植管理**: 作物选择和地块信息
- **市场分析**: 价格走势和市场预测
- **设施管理**: 大棚状态和升级选项
- **财务报告**: 年度经营数据和利润分析

### 游戏控制
- **暂停/继续**: 暂停或恢复游戏进程
- **游戏速度**: 1x/2x/4x 调节游戏速度
- **标签切换**: 在不同管理界面间切换

## 🏆 评分系统

最终得分基于以下因素：
- **总利润**: 7年经营期间的累计净利润
- **利润率**: 平均利润率超过20%有额外加成
- **可持续性**: 合理的轮作和土壤管理
- **风险应对**: 成功应对随机事件的能力

## 🛠️ 技术栈

- **前端框架**: Vanilla JavaScript (ES6+)
- **构建工具**: Vite
- **图形渲染**: Canvas API
- **样式**: CSS3 (Flexbox + Grid)
- **模块化**: ES6 Modules

## 🔧 开发指南

### 添加新作物
1. 在 `CropSystem.js` 的 `initializeCrops()` 中添加作物定义
2. 在 `Renderer.js` 中添加作物颜色和图标
3. 更新经济系统的价格设置

### 添加新事件
1. 在 `RandomEventSystem.js` 的 `initializeEvents()` 中定义事件
2. 设置触发概率和效果参数
3. 在UI中添加事件显示逻辑

### 自定义样式
- 修改 `style.css` 中的CSS变量调整主题色彩
- 调整响应式断点适配不同屏幕尺寸
- 自定义组件样式和动画效果
